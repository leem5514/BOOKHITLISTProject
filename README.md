# 개요
+ 번개장터, 당근과 같은 중고거래 사이트들은 여러가지가 있다. 하지만 '도서'라는 카테고리만을 가지고 보여주는 도서중고거래사이트들은 존재하지 않는다.<br>
심화적으로 도서만을 위한 중고거래 와 커뮤니티 사이트를 구현해보고 싶었다.
